# 房子大了电话小了 applied

Applied biostatistics qualifying-exam practice materials covering the 2022, 2024, and 2025 project styles.

Website: <https://yl8270.github.io/applied-biostatistics-qe/>

## Contents

- [2022: Lung cancer pulmonary function and survival](2022/)
- [2024: NSQIP-style bariatric surgery outcomes](2024/)
- [2025: Kidney transplant graft survival](2025/)
- [Applied biostatistics research-to-report handbook](Applied_Biostatistics_Research_to_Report_Handbook_Package/)
- [Three-year textbook and training package](Applied_Biostatistics_Three_Year_Textbook_Training_Package/)

The datasets included in the year-specific projects are synthetic practice data. They are not real clinical datasets and must not be interpreted as clinical evidence.

The three original qualifying-exam documents are intentionally excluded from this public repository. This repository contains independent practice code, synthetic data, reports, tables, figures, and study materials.
